# Stative Verbs — Конспект

**Stative verbs** (глаголы состояния) — это глаголы, которые описывают состояние, а не действие. Обычно они **не употребляются** в временах группы Continuous.

## Список основных стативных глаголов

| Эмоции и предпочтения | Мышление и мнение | Принадлежность и наличие | Восприятие | Другие |
|----------------------|------------------|-------------------------|-----------|--------|
| like | know | belong | see | seem |
| love | realise | fit | hear | depend |
| hate | suppose | contain | sound | matter |
| want | mean | consist | taste | concern |
| need | understand | appear | smell | involve |
| prefer | believe | own | look (= seem) | include |
| agree | remember | possess | | lack |
| mind | recognise | owe | | measure* |
| wish | imagine | | | weigh* |
| dislike | think** | | | be |
| doubt | feel** | | | have*** |
| deserve | | | | |

*measure/weigh* — только в значении «иметь длину/вес»  
*think/feel* — только в значении «иметь мнение»  
*have* — только в значении «владеть»

## Stative vs. Dynamic Verbs

Stative verb = состояние (не используется в Continuous)
Dynamic verb = действие (можно использовать в Continuous)

### Пример:
|           Неправильно         |      Правильно        |
|---------------|---------------|
| ~~I am knowing the answer~~   | I know the answer     |
| ~~She is wanting some water~~ | She wants some water  |
| ~~This soup is tasting good~~ | This soup tastes good |

## Глаголы с двойным значением

Некоторые глаголы могут быть **и стативными, и динамичными** в зависимости от контекста.

### BE
| Значение | Тип | Пример |
|----------|-----|--------|
| Быть (качество личности) | Stative | You **are** stupid. *(ты вообще глупый)* |
| Вести себя / действовать | Dynamic | You **are being** stupid. *(ты ведёшь себя глупо сейчас)* |

### THINK
| Значение | Тип | Пример |
|----------|-----|--------|
| Иметь мнение | Stative | I **think** coffee is great. |
| Размышлять, обдумывать | Dynamic | What **are** you **thinking** about? |

### HAVE
| Значение | Тип | Пример |
|----------|-----|--------|
| Владеть, иметь | Stative | I **have** a car. |
| Участвовать в действии | Dynamic | I'**m having** a party / a bath / a good time. |

### TASTE / SMELL / FEEL / SEE
| Значение | Тип | Пример |
|----------|-----|--------|
| Обладать вкусом/запахом/ощущением | Stative | This soup **tastes** great. |
| Действие: пробовать/нюхать/трогать | Dynamic | The chef **is tasting** the soup. |

**Правило-подсказка**:  
Если глагол описывает **неизменное состояние** → stative.  
Если глагол описывает **процесс или действие** → dynamic.

## Быстрая проверка: можно ли использовать в Continuous?

Знаю → know → I am knowing
Люблю → love → I am loving* (*кроме рекламы)
Понимаю → understand → I am understanding
Принадлежит → belong → It is belonging
Кажется → seem → It is seeming

**Совет**: При сомнении — спросите себя: *«Это действие, которое можно увидеть в процессе, или состояние?»*  
Если состояние → скорее всего, это stative verb → избегайте Continuous.
