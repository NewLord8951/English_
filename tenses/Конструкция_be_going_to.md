Конструкция to be going to
В каких ситуациях мы используем эту конструкцию?
1.	Для описания запланированных или намеренных действий в будущем, даже если они еще не организованны. (собираюсь)
2.	Для предсказания бущих событий на основе данных. (I feel terrible. I think I’m going to be sick. Я чувствую себя ужасно, наверное, заболеваю)  
3.	Для описания не свершившихся запланированных действий в прошлом. (We were going to travel by train, but then we decided to go by car instead.)
В неформальной речи можно встретить сокращение конструкции to be going to до to be gonna. 
Для описания планов в будущем так же можно использовать Present Continuous. 
Как мы образуем положительное предложение с конструкцией to be going to?
Настоящее:
1.	I + am + going to + глагол
2.	He, She, It + is + going to + глагол
3.	We, They, You, We + are + going to + глагол
Прошедшее:
1.	I, He, She, It + was + going to + глагол
2.	 We, They, You, We + were + going to + глагол
Как мы образуем отрицание с конструкцией to be going to?
Настоящее:
1.	I + am not + going to + глагол
2.	He, She, It + is not (isn’t) + going to + глагол
3.	We, They, You, We + are not (aren’t) + going to + глагол
Прошедшее:
1.	I, He, She, It + was not (wasn’t) + going to + глагол
2.	We, They, You, We + were not (weren’t) + going to + глагол
Как мы образуем вопрос с конструкцией to be going to?
Настоящее:
1.	Am + I + going to + глагол
2.	Is + He, She, It + going to + глагол
3.	Are + We, They, You, We + going to + глагол
Прошедшее:
1.	Was + I, He, She, It + going to + глагол
2.	 Were + We, They, You, We + going to + глагол
